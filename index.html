Why Speech Emotion Recognition?

Human communication conveys emotion through tone, pitch, rhythm, and energy â€” not just words. Current voice interfaces (Alexa, Siri) respond only to what you say, not how you feel. Speech Emotion Recognition (SER) bridges this gap by automatically classifying emotional states directly from audio signals.

Research Gap / Our Innovation

1
OOP Pipeline Architecture

Modular classes (AudioProcessor â†’ FeatureExtractor â†’ EmotionClassifier) make it easy to swap feature methods or models without rewriting the pipeline â€” unlike most tutorial-style scripts.

2
Combined Feature Vector

We combine MFCC + delta + delta-delta + pitch + energy + ZCR into a 247-dimensional vector. Most student projects use only plain MFCC mean, losing temporal dynamics captured by delta features.

3
Extensible Roadmap

The same pipeline is designed to accept deep learning models in Semester 3 â€” simply swap the classifier stage with a 1D-CNN or LSTM without changing preprocessing or feature extraction.

class AudioProcessor:
    def full_preprocess(self, path):
        signal, sr = self.load(path)
        signal = self.remove_silence(signal)
        signal = self.normalize(signal)
        signal = self.apply_preemphasis(signal)
        return signal, sr

    def normalize(self, signal):
        return signal / np.max(np.abs(signal))
src/feature_extraction.py 
class FeatureExtractor:
    def extract_all(self, signal, sr):
        mfcc   = librosa.feature.mfcc(
                   y=signal, sr=sr, n_mfcc=40)
        delta  = librosa.feature.delta(mfcc)
        delta2 = librosa.feature.delta(mfcc,order=2)
        # combine â†’ 247-dim vector
        return np.concatenate([
            mean_std(mfcc), mean_std(delta),
            mean_std(delta2), pitch, energy, zcr
        ])

class EmotionClassifier:
    def __init__(self, model_type='random_forest'):
        self.pipeline = Pipeline([
            ('scaler', StandardScaler()),         # normalise features
            ('clf',    MODEL_REGISTRY[model_type]) # swap any sklearn model
        ])

    def train(self, X, y):
        y_enc = self.label_encoder.fit_transform(y)
        self.pipeline.fit(X, y_enc)

    def predict(self, X):
        return self.label_encoder.inverse_transform(
                   self.pipeline.predict(X))
